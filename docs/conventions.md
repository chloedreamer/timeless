# Conventions: Timeless

## Màu sắc
- Rose/hồng nhạt: tone chính — sang trọng, nữ tính
- Gold (#b8976a): accent — giá tiền, điểm nhấn
- Không dùng màu sắc loè loẹt

## Font
- System font stack (không import Google Fonts)
- 4 kích thước cố định: xl (1.8rem), l (15px), m (13px), s (12px)

## Emoji phân loại nhóm
- 🌸 = nhóm thương hiệu chính (Joie des Roses, By Cún, méli...)
- 🔴 = nhóm thương hiệu khác (LSOUL, ONONMM, VEOS...)

## Định dạng giá
- Luôn dùng dấu chấm ngăn cách hàng nghìn: `550.000₫`
- Ký hiệu đồng: `₫` (không phải VND hay đ)

## Comment code
- Viết bằng tiếng Việt
